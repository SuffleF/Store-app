### Setup .env file

```js
PORT=3000

PGUSER=...
PGPASSWORD=...
PGHOST=...
PGDATABASE=...

ARCJET_KEY=...
ARCJET_ENV=development
```

### Run the backend

```shell
npm run dev
```

### Run the frontend

```shell
cd frontend
npm run dev
```
